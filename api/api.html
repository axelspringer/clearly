Template

Endpoint to work with the data model (template). A template is a structure which combines meta types and content types into a managable entity. Templates can be aggregated by channels.

/template/{templateId} get

A specific template, identified by its id

get

Retrieve the template with id

get /template/{templateId}

Retrieve the template with id

  • Request
  • Response

URI Parameters

  • templateId: required (string)

    The id of the template to retrieve

    Example:

    123

Query Parameters

  • callback: (string)

    Callback function name. All output will be wrapper under this function name.

    Example:

    callbackFunction
  • count: (integer)

    Number of items you would like to receive.

    Example:

    1

HTTP status code 200

Schema:

  "$schema": "http://json-schema.org/draft-03/schema",

      "type": "boolean" ,

      "type": "integer" ,
